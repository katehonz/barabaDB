# B-Tree индекс

Упорядоченная индексная структура для эффективных диапазонных сканирований и точечных поисков.

## Использование

```nim
import barabadb/storage/btree

var btree = newBTreeIndex[string, string]()

btree.insert("key1", "value1")
btree.insert("key2", "value2")

let values = btree.get("key1")

let range = btree.scan("key_a", "key_z")

btree.delete("key1")
```

## Функции

- Упорядоченное хранение ключ-значение
- Диапазонные запросы (BETWEEN, >, <, >=, <=)
- Префиксные сканирования
- Настраиваемый размер страницы
- Поддержка итераторов

## Сценарии использования

- Первичные ключевые индексы
- Вторичные индексы для часто запрашиваемых столбцов
- Данные с диапазонным разбиением
